## Pokémon Capture Rate Prediction

### What is this? 🤔
This project applies machine learning regression algorithms using Python to a Pokémon dataset available on Kaggle.

The goal of this project is to compare different regression models and identify the best-performing one in predicting the capture rate of Pokémon based on multiple attributes such as attack, defense, speed, height, weight, generation, type, and other statistical characteristics.
