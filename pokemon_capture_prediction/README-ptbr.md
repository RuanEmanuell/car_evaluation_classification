## Predição da Taxa de Captura de Pokémon

### O que é isso? 🤔
Este projeto aplica algoritmos de regressão de machine learning utilizando Python em um dataset de Pokémon disponível no Kaggle.

O objetivo deste projeto é comparar diferentes modelos de regressão e identificar aquele com melhor desempenho na predição da taxa de captura dos Pokémon, com base em múltiplos atributos como ataque, defesa, velocidade, altura, peso, geração, tipo e outras características estatísticas.
