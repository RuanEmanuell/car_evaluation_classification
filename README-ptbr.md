## Classificação de Avaliação de Carros

### O que é isso? 🤔 
Este projeto aplica algoritmos de classificação de machine learning utilizando Python no dataset "Car Evaluation", disponível no Repositório de Machine Learning da UCI.
O objetivo deste projeto é comparar diferentes modelos de classificação e identificar aquele com o melhor desempenho na predição da aceitabilidade de um carro com base em múltiplas características categóricas.
